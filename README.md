# Document Intelligence System

A rule-based NLP application that allows users to upload PDF, Excel, or CSV files and query them using natural language.

## Features
- PDF, Excel, CSV support
- Spell correction
- Intent detection
- Summary, Search, Filter, Count, Aggregation, Sorting
- Structured output

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Sample Queries
- Summarize this document
- List all employees from Mumbai
- Find expenses above 5000
- How many products are out of stock
